# DEVICE TWEAKER

**DEVICE TWEAKER** — утилита для Windows 11, объединяющая MSI Utility V3, Interrupt Affinity Policy Tool, ReservedCpuSets и управление IMOD в более удобном интерфейсе с интеллектуальной авто‑оптимизацией.
